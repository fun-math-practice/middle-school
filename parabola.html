📖 Part 1: Key Vocabulary Video 1
A quadratic equation looks like y = ax² + bx + c — notice the x²! That's what makes it different from a linear equation.
Its graph is a curved U-shape called a parabola. Here are the key parts:
Vertex
The tip of the parabola — either the lowest point (opens up) or highest point (opens down).
Axis of Symmetry
A vertical line through the vertex. The two halves of the parabola are mirror images.
Opens Up / Opens Down
If a > 0 → opens up (U shape). If a < 0 → opens down (∩ shape).
x-intercepts (Roots)
Where the parabola crosses the x-axis. A parabola can have 0, 1, or 2 of these.

🔀 Part 5: The Three Forms + New Video
▶ Watch First
Standard, Vertex & Factored Forms of a Quadratic
Explains all three forms and what each one tells you · youtube.com/watch?v=8jtoCo8Jk0Q
▶
The same parabola can be written three different ways. Each form makes certain information easy to read off:
① Standard Form (Expanded)
y = ax² + bx + c
✔ c is the y-intercept instantly  |  ✔ Most common form  |  ✘ Vertex takes a formula
Example: y = x² − 2x − 3 → y-intercept = −3
② Vertex Form
y = a(x − h)² + k
✔ Vertex is (h, k) instantly  |  ✔ Easy to graph  |  ✘ x-intercepts take work
Example: y = (x − 1)² − 4 → vertex = (1, −4)
③ Factored Form (Intercept Form)
y = a(x − r)(x − s)
✔ x-intercepts are r and s instantly  |  ✔ Roots visible  |  ✘ Not always possible
Example: y = (x + 1)(x − 3) → x-intercepts = −1 and 3
💡 All three forms describe the exact same parabola!   y = x²−2x−3  =  y = (x−1)²−4  =  y = (x+1)(x−3)
Question 16 · Multiple Choice
Which form is y = 2(x − 3)² + 5?
Look for the pattern y = a(x − h)² + k. Here a = 2, h = 3, k = 5. That's vertex form — and the vertex is (3, 5)!
Question 17 · Multiple Choice
The equation y = (x + 4)² − 7 is in vertex form. What is the vertex?
In y = (x − h)² + k, notice the minus sign. Here we have (x + 4) which is (x − (−4)), so h = −4. And k = −7. Vertex = (−4, −7).
Question 18 · Multiple Choice
From the factored form y = (x − 2)(x + 5), what are the x-intercepts?
Set each factor to zero. (x − 2) = 0 → x = 2. (x + 5) = 0 → x = −5. The x-intercepts are where y = 0, so x = 2 and x = −5.
Question 19 · Multiple Choice
In standard form y = ax² + bx + c, which value gives you the y-intercept directly?
Plug in x = 0: y = a(0)² + b(0) + c = c. So the y-intercept is always c — the constant at the end!
Question 20 · Fill In

The two x-intercepts are x = ___ and x = ___.

(Midpoint = add them together and divide by 2)

(x − 3) = 0 → x = 3. (x + 1) = 0 → x = −1. Midpoint = (3 + (−1)) / 2 = 2 / 2 = 1. So vertex x-coordinate = 1.
Question 21 · Multiple Choice
You want to find the highest point of a parabola as quickly as possible. Which form is most useful?
The "highest point" is the vertex! Vertex form gives you (h, k) — the vertex — directly. No formula needed.
