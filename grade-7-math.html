Core Concept

Lesson 1.1 · Integers & Absolute Value

An integer is any whole number, its opposite, or zero: … −3, −2, −1, 0, 1, 2, 3 …

The absolute value of a number is its distance from zero on a number line — distance is always non-negative.

Key rule:
|positive| = positive  ·  |negative| = positive  ·  |0| = 0
e.g. |−7| = 7 and |+7| = 7 → opposites share the same absolute value

Lesson 1.2 · Comparing & Ordering Rational Numbers

A rational number is any number writable as a fraction p/q (q ≠ 0). This includes integers, fractions, and terminating/repeating decimals.

Three rules for comparing:
1. On a number line, numbers get larger going right →
2. To compare fractions, find a common denominator first
3. For negatives: less negative = larger  (−1 > −3, because −1 is further right)

Example: order −3/2, −1/2, 0, 3/4, 5/4 → convert to decimals: −1.5, −0.5, 0, 0.75, 1.25

Lesson 1.3 · Additive Inverse

Two numbers are additive inverses if they add up to exactly zero.

How to find the additive inverse: flip the sign!
Additive inverse of +6.5 is −6.5  because  6.5 + (−6.5) = 0
Additive inverse of −3/4 is +3/4  because  −3/4 + 3/4 = 0
Special: 0 is its own additive inverse — 0 + 0 = 0

Lesson 1.4 · Adding Integers

Same signs: Add the absolute values, keep that sign.
Different signs: Subtract the smaller absolute value from the larger; keep the sign of whichever had the bigger absolute value.

Same signs: −2 + (−3) → both negative → 2+3=5 → −5
Diff. signs: −3 + 5 → subtract: 5−3=2 → 5 is bigger & positive → +2

Lesson 1.5 · Subtracting Integers

Key rule: Subtracting is the same as adding the opposite.

Rewrite rule: a − b = a + (−b)

Example 1: 4 − 7  = 4 + (−7) = −3
Example 2: 4 − (−7) = 4 + (+7) = 11  (subtracting a negative = adding a positive!)
Example 3: −5 − (−3) = −5 + 3 = −2

Lesson 1.6 · Multiplying & Dividing Integers

The sign rules are the same for both × and ÷ — only the signs matter:

Same signs → result is POSITIVE
Different signs → result is NEGATIVE

(+3)×(+4) = +12  ·  (−3)×(−4) = +12
(+3)×(−4) = −12  ·  (−3)×(+4) = −12

Core Concept

Lesson 1.7 · Adding & Subtracting Rational Numbers

Fractions: You must have a common denominator before adding or subtracting.

3 Steps for fractions:
Step 1: Find the LCD (Least Common Denominator)
Step 2: Convert each fraction to an equivalent fraction with that LCD
Step 3: Add or subtract the numerators only — the denominator stays the same

Worked example: −1/2 + 3/4
LCD(2, 4) = 4  →  −1/2 becomes −2/4
−2/4 + 3/4 = 1/4

Decimals: Line up the decimal points, then add/subtract. Apply integer sign rules to the result.

Worked example: −3.6 + 1.25
Align: −3.60 and +1.25 → different signs → subtract: 3.60 − 1.25 = 2.35
Larger absolute value is 3.60 (negative) → answer: −2.35

Core Concept

Lesson 1.8 · Multiplying & Dividing Rational Numbers

Multiply fractions: numerator × numerator over denominator × denominator. Simplify after.

Divide fractions: Keep the first fraction, Flip the second (its reciprocal), then Multiply. (Keep–Change–Flip)

Signs: Same signs → positive result. Different signs → negative result.

Core Concept

Lesson 1.9 · Fractions & Decimals

Fraction → Decimal: Divide the numerator by the denominator.

Terminating: 3/4 → 3 ÷ 4 = 0.75  (remainder reaches 0 — it stops)
Repeating: 1/3 → 1 ÷ 3 = 0.333…  (pattern repeats forever, written 0.3̄)

Decimal → Fraction: Write digits over a power of 10, then simplify.
e.g. 0.6 = 6/10 = 3/5  ·  0.25 = 25/100 = 1/4  ·  0.125 = 125/1000 = 1/8

Core Concept

Lesson 1.10 · Mathematical Properties

These properties let you rearrange and regroup numbers — they work for all rational numbers and make complex calculations much easier.

Commutative  a + b = b + a  (swap the order)
e.g. −1 + 11 = 11 + (−1) = 10

Associative  (a + b) + c = a + (b + c)  (change the grouping)
e.g. (2 + 3) + 4 = 2 + (3 + 4) = 9

Strategy tip: Look for additive inverse pairs that cancel to zero, or group numbers that make tidy sums.

Core Concept

Lesson 1.11 · Real World Problems

The key skill is translating words into math expressions with the correct signs.

Common sign translations:
Debt / spent / withdrew / dropped / lost → negative (−)
Earned / deposited / gained / rose / found → positive (+)

Strategy:
1. Identify what each number represents
2. Assign the correct sign
3. Write the expression & calculate
4. Ask: does the answer make sense in context?
